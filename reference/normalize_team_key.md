# Normalize the team key

Normalize the team key

## Usage

``` r
normalize_team_key(team)
```

## Arguments

- team:

  integer ID (e.g., 21), character full name (e.g., 'Colorado
  Avalanche'), OR three-letter code (e.g., 'COL')

## Value

integer in 1:68, character full name, OR three-letter code
