# Build cached xG booster path

`.xg_cached_booster_path()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_cached_booster_path(path, bundle = .xg_load_bundle())
```

## Arguments

- path:

  character file or bundle path

- bundle:

  xG model bundle

## Value

Internal helper output.
