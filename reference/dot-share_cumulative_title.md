# Build a cumulative xG title

`.share_cumulative_title()` builds the title for a game-level cumulative
xG time series.

## Usage

``` r
.share_cumulative_title(labels)
```

## Arguments

- labels:

  list returned by
  [`.share_game_labels()`](https://rentosaijo.github.io/nhlscraper/reference/dot-share_game_labels.md)

## Value

character scalar
