# Read lookup column with fallback

`.shift_situation_lookup_col()` is an internal helper for
[`calculate_shift_times_by_situation()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_shift_times_by_situation.md).

## Usage

``` r
.shift_situation_lookup_col(source, col, fallback)
```

## Arguments

- source:

  data.frame lookup source

- col:

  character column name

- fallback:

  fallback value

## Value

Internal helper output.
