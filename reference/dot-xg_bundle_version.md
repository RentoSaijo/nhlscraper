# Build xG bundle cache version

`.xg_bundle_version()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_bundle_version(bundle = .xg_load_bundle())
```

## Arguments

- bundle:

  xG model bundle

## Value

Internal helper output.
