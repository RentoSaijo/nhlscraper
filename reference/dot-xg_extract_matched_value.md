# Extract matched skater-slot value

`.xg_extract_matched_value()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_extract_matched_value(id_mat, value_mat, player_id)
```

## Arguments

- id_mat:

  integer skater ID matrix

- value_mat:

  numeric skater value matrix

- player_id:

  integer player ID vector

## Value

Internal helper output.
