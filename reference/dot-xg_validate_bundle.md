# Validate xG model bundle

`.xg_validate_bundle()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_validate_bundle(bundle)
```

## Arguments

- bundle:

  xG model bundle

## Value

Internal helper output.
