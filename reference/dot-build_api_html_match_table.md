# Build API-side HTML matching features

`.build_api_html_match_table()` derives the API-side matching features
used to align HTML play-by-play rows back to API play-by-play rows.

## Usage

``` r
.build_api_html_match_table(play_by_play)
```

## Arguments

- play_by_play:

  data.frame play-by-play object

## Value

data.frame of API-side matching features
