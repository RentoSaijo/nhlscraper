# Draw the shot-map legends

`.draw_share_shot_legends()` draws outcome symbols and the xG color ramp
above the rink.

## Usage

``` r
.draw_share_shot_legends(scale, spec)
```

## Arguments

- scale:

  list returned by
  [`.share_xg_scale()`](https://rentosaijo.github.io/nhlscraper/reference/dot-share_xg_scale.md)

- spec:

  list returned by
  [`.share_plot_spec()`](https://rentosaijo.github.io/nhlscraper/reference/dot-share_plot_spec.md)
