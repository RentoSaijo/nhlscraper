# Fetch shift data for xG features

`.xg_fetch_shift_data()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_fetch_shift_data(game_ids)
```

## Arguments

- game_ids:

  integer game ID vector

## Value

Internal helper output.
