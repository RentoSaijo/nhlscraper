# Validate xG booster path

`.xg_validate_booster_path()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_validate_booster_path(path)
```

## Arguments

- path:

  character file or bundle path

## Value

Internal helper output.
