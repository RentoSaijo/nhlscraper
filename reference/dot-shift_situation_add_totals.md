# Add situation total columns

`.shift_situation_add_totals()` is an internal helper for
[`calculate_shift_times_by_situation()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_shift_times_by_situation.md).

## Usage

``` r
.shift_situation_add_totals(summary, overlaps)
```

## Arguments

- summary:

  data.frame summary input

- overlaps:

  data.frame overlap totals

## Value

Internal helper output.
