# Convert to the appropriate team three-letter code

Convert to the appropriate team three-letter code

## Usage

``` r
to_team_tri_code(team, lookup = .to_team_tri_code)
```

## Arguments

- team:

  integer ID (e.g., 21), character full name (e.g., 'Colorado
  Avalanche'), OR three-letter code (e.g., 'COL')

## Value

three-letter code
