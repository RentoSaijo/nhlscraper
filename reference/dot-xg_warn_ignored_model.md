# Warn about ignored xG model argument

`.xg_warn_ignored_model()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_warn_ignored_model(model, fn_name)
```

## Arguments

- model:

  legacy model argument

- fn_name:

  character caller name

## Value

Internal helper output.
