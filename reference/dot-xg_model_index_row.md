# Find xG model index row

`.xg_model_index_row()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_model_index_row(model_key, bundle = .xg_load_bundle())
```

## Arguments

- model_key:

  character xG model key

- bundle:

  xG model bundle

## Value

Internal helper output.
