# Download xG booster file

`.xg_download_booster()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_download_booster(row, bundle = .xg_load_bundle())
```

## Arguments

- row:

  data.frame model-index row

- bundle:

  xG model bundle

## Value

Internal helper output.
