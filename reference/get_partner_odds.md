# Get the real-time game odds for a country by partnered bookmaker

`get_partner_odds()` is deprecated. Use
[`game_partner_odds()`](https://rentosaijo.github.io/nhlscraper/reference/game_partner_odds.md)
instead.

## Usage

``` r
get_partner_odds(country = "US")
```

## Arguments

- country:

  two-letter code (e.g., 'CA'); see
  [`countries()`](https://rentosaijo.github.io/nhlscraper/reference/countries.md)
  for reference

## Value

data.frame with one row per game
