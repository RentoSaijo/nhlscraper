# Build a shot-location title

`.share_shot_location_title()` builds the title for a team-specific shot
map.

## Usage

``` r
.share_shot_location_title(labels, team)
```

## Arguments

- labels:

  list returned by
  [`.share_game_labels()`](https://rentosaijo.github.io/nhlscraper/reference/dot-share_game_labels.md)

- team:

  character of `home` or `away`

## Value

character scalar
