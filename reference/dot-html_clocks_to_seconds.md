# Convert clocks to seconds

`.html_clocks_to_seconds()` vectorizes
[`.html_time_to_seconds()`](https://rentosaijo.github.io/nhlscraper/reference/dot-html_time_to_seconds.md)
for shift report clock columns.

## Usage

``` r
.html_clocks_to_seconds(x)
```

## Arguments

- x:

  character vector

## Value

integer vector
