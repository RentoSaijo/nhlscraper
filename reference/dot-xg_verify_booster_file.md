# Verify xG booster file checksum

`.xg_verify_booster_file()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_verify_booster_file(path, row)
```

## Arguments

- path:

  character file or bundle path

- row:

  data.frame model-index row

## Value

Internal helper output.
