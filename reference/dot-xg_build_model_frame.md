# Build xG model frame

`.xg_build_model_frame()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_build_model_frame(shots, play_by_play)
```

## Arguments

- shots:

  data.frame shot rows

- play_by_play:

  data.frame play-by-play input

## Value

Internal helper output.
