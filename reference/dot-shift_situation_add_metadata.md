# Add team and player metadata

`.shift_situation_add_metadata()` is an internal helper for
[`calculate_shift_times_by_situation()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_shift_times_by_situation.md).

## Usage

``` r
.shift_situation_add_metadata(summary, roster = NULL)
```

## Arguments

- summary:

  data.frame summary input

- roster:

  optional data.frame roster input

## Value

Internal helper output.
