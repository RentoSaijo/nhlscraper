# Extract skater slot indices

`.xg_extract_slot_indices()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_extract_slot_indices(data, suffix)
```

## Arguments

- data:

  data.frame feature input

- suffix:

  character skater-slot column suffix

## Value

Internal helper output.
