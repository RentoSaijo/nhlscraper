# Call the ESPN API with 429 (rate limit) error-handling

Call the ESPN API with 429 (rate limit) error-handling

## Usage

``` r
espn_api(path, query = list(), type)
```

## Arguments

- path:

  character

- query:

  list

- type:

  character of 'g' for general and 'c' for core

## Value

parsed JSON (i.e., data.frame or list)
