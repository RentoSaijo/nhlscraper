# Build xG model key

`.xg_model_key()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_model_key(target_season, partition)
```

## Arguments

- target_season:

  integer target season

- partition:

  character partition code

## Value

Internal helper output.
