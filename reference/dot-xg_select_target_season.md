# Select xG target season

`.xg_select_target_season()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_select_target_season(game_id, bundle = .xg_load_bundle())
```

## Arguments

- game_id:

  integer game ID vector

- bundle:

  xG model bundle

## Value

Internal helper output.
