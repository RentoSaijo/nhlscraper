# Normalize xG shot type

`.xg_normalize_shot_type()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_normalize_shot_type(x)
```

## Arguments

- x:

  vector input

## Value

Internal helper output.
