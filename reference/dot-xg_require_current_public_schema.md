# Require current public play-by-play schema

`.xg_require_current_public_schema()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_require_current_public_schema(play_by_play)
```

## Arguments

- play_by_play:

  data.frame play-by-play input

## Value

Internal helper output.
