# Normalize xG missed-shot reason

`.xg_normalize_missed_reason()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_normalize_missed_reason(x)
```

## Arguments

- x:

  vector input

## Value

Internal helper output.
