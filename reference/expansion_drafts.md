# Access all the expansion drafts

`expansion_drafts()` retrieves all the expansion drafts as a
`data.frame` where each row represents expansion draft and includes
detail on date/season filtering windows and chronological context plus
reference definitions and rules-framework information.

## Usage

``` r
expansion_drafts()
```

## Value

data.frame with one row per expansion draft

## Examples

``` r
all_expansion_drafts <- expansion_drafts()
```
