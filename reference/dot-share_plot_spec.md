# Build share-plot output settings

`.share_plot_spec()` returns file naming, canvas size, and legend
offsets for an IG or X plot.

## Usage

``` r
.share_plot_spec(platform, plot_type)
```

## Arguments

- platform:

  character of `ig` or `x`

- plot_type:

  character of `shots` or `cumulative`

## Value

named list of plot settings
