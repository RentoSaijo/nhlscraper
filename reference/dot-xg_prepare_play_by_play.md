# Prepare play-by-play for xG scoring

`.xg_prepare_play_by_play()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_prepare_play_by_play(play_by_play)
```

## Arguments

- play_by_play:

  data.frame play-by-play input

## Value

Internal helper output.
