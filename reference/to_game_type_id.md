# Convert to the appropriate game type ID

Convert to the appropriate game type ID

## Usage

``` r
to_game_type_id(game_type)
```

## Arguments

- game_type:

  integer in 1:3 (where 1 = pre-season, 2 = regular season, 3 =
  playoff/post-season) OR character of 'pre', 'regular', or
  'playoff'/'post'

## Value

integer in 1:3
