# Build skater-slot matrix

`.xg_build_skater_matrix()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_build_skater_matrix(data, suffix, mode = "numeric")
```

## Arguments

- data:

  data.frame feature input

- suffix:

  character skater-slot column suffix

- mode:

  character matrix storage mode

## Value

Internal helper output.
