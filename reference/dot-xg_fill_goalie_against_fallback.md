# Fill goalie-against fallback columns

`.xg_fill_goalie_against_fallback()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_fill_goalie_against_fallback(play_by_play)
```

## Arguments

- play_by_play:

  data.frame play-by-play input

## Value

Internal helper output.
