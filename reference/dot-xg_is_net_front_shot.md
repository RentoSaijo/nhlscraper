# Flag net-front shots

`.xg_is_net_front_shot()` is an internal helper for
[`calculate_expected_goals()`](https://rentosaijo.github.io/nhlscraper/reference/calculate_expected_goals.md).

## Usage

``` r
.xg_is_net_front_shot(x_coord_norm, abs_y_coord_norm)
```

## Arguments

- x_coord_norm:

  numeric normalized x-coordinate vector

- abs_y_coord_norm:

  numeric absolute normalized y-coordinate vector

## Value

Internal helper output.
