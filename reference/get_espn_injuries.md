# Access the real-time ESPN injury reports

`get_espn_injuries()` is deprecated. Use
[`espn_injuries()`](https://rentosaijo.github.io/nhlscraper/reference/espn_injuries.md)
instead.

## Usage

``` r
get_espn_injuries()
```

## Value

nested data.frame with one row per team (outer) and player (inner)
